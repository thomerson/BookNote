# 第一章 信息化和信息系统

# 信息
* 客观事物，用来消除不确定性
* 单位：bit
* 特征
    * 客观性：主观信息必然转变为客观信息
    * 传递性：时间传递就是存储，空间上传递是转移或者扩散

* 7个质量属性(安全靠精整及经验)
    * 精准性
    * 完整性
    * 可靠性
    * 及时性
    * 经济性
    * 可验证性
    * 安全性

* 信息的传输模型
![信息传输模型](..\source\images\信息传输模型.PNG)
    * 信源
    * 编码
    * 信道
    * 解码(最容易出错)
    * 信宿
    * 噪声(所有环节都有可能存在，信道中存在最多)
    * 提高可靠性，增加冗余(重要的事情说三遍)
