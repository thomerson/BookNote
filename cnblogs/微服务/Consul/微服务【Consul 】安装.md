## Windows 下安装

1. 首先在consul.exe文件目录下的地址栏中输入cmd


2. 接着输入consul指令，敲击回车安装consul

3. 输入```consul agent -dev```指令，敲击回车启动consul

4. 打开```http://localhost:8500/```
