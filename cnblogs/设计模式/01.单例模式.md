## 单例模式

* 它只有一个实例

* 向外提供访问点

* 考虑到多线程情况下创建实例


### 分类

* 懒汉式

不支持多线程

* 

```c#

using System;

namespace 单例模式
{
    /// <summary>
    /// 它只有一个实例
    /// 向外提供访问点
    /// </summary>
    public class Signleton
    {
        private Signleton() { }

        private static Signleton instance;

        /// <summary>
        /// 多线程访问
        /// </summary>
        private static object locker = new object();

        public Signleton GetInstance()
        {
            if (instance == null)
            {
                lock (locker)
                {
                    if (instance == null)
                    {
                        instance = new Signleton();
                    }
                }
            }
            return instance;
        }
    }
}


```
