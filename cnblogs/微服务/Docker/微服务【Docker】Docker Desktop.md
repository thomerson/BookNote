## Docker Desktop

Docker桌面版,windows和mac

1. 准备

* windows10及以上系统

* BIOS开启虚拟化

* 启动```Hyper-V```服务

* WSL安装

### 功能

* images

    管理镜像

* containers    

    管理容器

* volumes

    管理数据卷
