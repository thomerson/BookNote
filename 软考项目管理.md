# 第一章 信息化和信息系统

# 信息
* 客观事物，用来消除不确定性
* 单位：bit
* 特征
    * 客观性：主观信息必然转变为客观信息
    * 传递性：时间传递就是存储，空间上传递是转移或者扩散

* 7个质量属性(安全靠精整及经验)
    * 精准性
    * 完整性
    * 可靠性
    * 及时性
    * 经济性
    * 可验证性
    * 安全性

* 信息的传输模型
![信息传输模型](..\source\images\信息传输模型.PNG)
    * 信源
    * 编码
    * 信道
    * 解码(最容易出错)
    * 信宿
    * 噪声(所有环节都有可能存在，信道中存在最多)
    * 提高可靠性，增加冗余(重要的事情说三遍)

# 信息系统

* 信息系统：输入数据，加工处理，产生信息的系统
* 信息化的五个层级
    * 产品信息化：智能家电，车载集成
    * 企业信息化：ERP,CRM
    * 产业信息化：资源，要素优化重组
    * 国民经济信息化：生产，流通，分配，消费四个环节信息闭环
    * 社会生活信息化：智慧城市，互联网金融
* 信息化基本内涵
    * 主体：全体社会成员
    * 时域：长期
    * 空域：一切领域
    * 手段：先进社会生产工具
    * 途径
    * 目标

* 国家信息化体系6要素⭐
    * 信息资源：核心人物
    * 信息网络：三网融合(电信网，广播电视网，计算机网络)
    * 信息技术应用：龙头
    * 信息技术和产业：物质基础
    * 信息化人才：成功之本  
    * 信息化政策法规和标准规范：保障
