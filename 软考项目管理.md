可重用代码估算
等价代码行=[(重新设计百分比+重新编码百分比+重新测试百分比)/3]*已有代码行

#### 参数估算

#### 三点估算
考虑估算中不确定性和风险

两种假定分布
* 三角分布
期望值=(最悲观+最乐观+最可能)/3
* 贝塔分布
期望值=(最悲观+最乐观+最可能*4)/6
* 标准差/标准偏差σ
反映平均值分散程度，表示精确度的重要指标
σ=(最悲观-最乐观)/6
方差v
v=σ²
   * ±1σ=68.26%
   * ±2σ=95.46%
   * ±3σ=99.73%
   * 两个路径的标准差：(σ1²+σ2²)开根号

成本类型⭐⭐⭐⭐⭐
* 直接成本：归属于项目的成本，例如差旅费，工资
* 间接成本：一般管理费用，或者多个项目分摊，例如管理费用，水电费，房租
* 固定成本：不变化的重复性成本呢，例如打印机的购买成本
* 可变成本：变动成本，例如打印纸张
* 全生命周期成本呢：设计生产运维等产品生命周期成本
* 沉默成本：已经发生的成本，与是否合理无关。决定是否继续某个出问题的项目时，不应该考虑沉没成本
* 机会成本：选择一个项目时而放弃另外的机会选择所带来的最高价值

#### 挣值管理⭐⭐⭐⭐⭐

#### 预测⭐⭐⭐
ETC完工尚需估算
EAC完工估算


EAC = AC + ETC

EAC = AC + (BAC - EV)/x

x = 1 非典型  剩余工作按照 **计划绩效**
x = CPI 典型 剩余工作按照 **当前绩效** **默认**
x = CPI * SPI 同时考虑CPI和SPI的影响
x = 新绩效 制定绩效
x = SPI 计算时间的EAC


#### 完工尚需绩效指数⭐⭐⭐⭐
TCPI完工尚需绩效指数 = 剩余工作的预算价值/剩余预算

TCPI = (BAC - EV)/(BAC - AC)  BAC可行
TCPI = (BAC - EV)/(EAC - AC)  BAC不可行用EAC代替

TCPI > 1 很难完成
TCPI = 1 正好完成
TCPI < 1 很容易完成  值越小越好

完工偏差VAC = BAC - EAC
VAC > 0 没有超过预算
VAC < 0 超过预算

#### 绩效审查⭐⭐⭐
偏差分析：比较计划活动成本与实际活动成本，识别成本基准与实际项目绩效之间的差异
趋势分析：判断绩效正在改善还是恶化
挣值绩效：实际进度及成本绩效与绩效测量基准进行比较

绩效审查
获取绩效->偏差分析->趋势分析->得到预测->判断是否变更

#### 一些财务概念
* 现值Present Value：未来的钱在现在的价值
* 折现率Discount Rate：将未来有限期预期收益折算成现值的比率
PV = FV(投资的终值)/(1 + r)n

* 净现值Net Present Value：时间段内报酬现值的总和减去原始投资额，**越大越好**
NPV > 0 接受该项目
NPV < 0 放弃该项目

多个备选方案的互斥决策中，净现值已经考虑了货币的时间价值，选择NPV正值中的最大者

* 内部收益率IRR(Internal Rate of return)  净现值等于零时的折现率，代表项目的抗风险（通货膨胀等）能力的大小，**越大越好**

* 静态投资回收期Payback Period:从项目投建之日起，用项目所得的净收益偿还原始投资额所需要的年限，**不考虑货币时间价值，越短越好**

* 投资回报率(Return On Investment):  年平均利润/投资总额 * 100%
不考虑货币时间价值，年平均利润是全部的利润。**越大越好**

* 效益成本比BCR(Bennfit-Cost Ratio)：项目投资与效益之间关系的比率 收益/投资
BCR > 1 接受该项目
BCR < 1 放弃该项目
**越大越好**
