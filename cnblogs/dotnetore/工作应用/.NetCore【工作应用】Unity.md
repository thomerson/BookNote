## Unity

Unity是一个[IoC]()容器，用来实现依赖注入（```Dependency Injection```，DI）,减少耦合


## Unity安装

```shell
install-package Unity
```

## 使用


### 注入方式

https://www.cnblogs.com/dotnet261010/p/9054201.html

* 构造函数注入

* 属性注入

* 方法注入

三种注入方式的执行顺序：**构造函数注入->属性注入->方法注入**

默认情况下如果构造函数上面没有使用特性，那么默认找**参数最多**的构造函数执行注入
